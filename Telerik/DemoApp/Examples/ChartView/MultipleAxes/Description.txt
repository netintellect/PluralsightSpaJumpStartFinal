RadCartesianChart provides built-in support for mutliple axes. Additional horizontal / vertical axes can be specified through the <b>CartesianSeries.HorizontalAxis / VerticalAxis</b> properties. You can control the location of each specific axis through the <b>Axis.HorizontalLocation / VerticalLocation</b> properties; also, you can control the axis colorization through the <b>Axis.ElementBrush</b> property.
