Create flexible and interactive layouts for your rich web applications with the new RadTileView. The control enables you to architect immersive user experiences with its configurable content "tiles", support for drag-and-drop rearrangement, and adjsutable position of the minimized items. RadTileView comes with built-in item animations, which are also fully customizeable to suit various scenarios. Also take advantage of the UI Virtualization if you deal with a lot of tiles.
