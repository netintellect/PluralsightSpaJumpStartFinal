Telerik ComboBox can be bound to various datasource types. This example shows how the combobox items' text can set using the <b>DisplayMemberPath</b> property or custom <b>ItemTemplate</b>. In addition you could define a separate template for the selected item using the <b>SelectionBoxItemTemplate</b> property.

You could configure the text search to use a specific property of your data items with the telerik:TextSearch.TextPath attached property.
